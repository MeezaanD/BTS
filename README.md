# BTS 2026 Comeback Countdown

A beautiful, interactive countdown application celebrating BTS's official reunion. With BTS beginning their military discharge in December 2024, this app marks the anticipated reunion date of **March 20, 2026**.

## Features

- **Live Countdown Timer** - Real-time countdown to March 20, 2026, showing days, hours, minutes, and seconds
- **Member Gallery** - Elegant member cards displaying all 7 BTS members with smooth animations
- **Immersive Design** - Stunning purple-themed UI with animated stars, gradient effects, and stage lighting
- **ARMY Messages** - Inspirational messages for the ARMY fandom
- **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **No External APIs** - Pure frontend experience with zero dependencies on external services

## Tech Stack

- **React 19** - Modern UI framework with hooks
- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
